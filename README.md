# Cordillera skeleton
Example of skeleton application powered by Cordillera Framework

**REQUIREMENTS**

* PHP >=5.4
* ext-mcrypt
* ext-json
* Mysql 5.1.X (Just for the example application)

**Startup**

	$ git clone https://github.com/rad8329/Cordillera-skeleton.git
	$ cd Cordillera-skeleton
	$ vagrant up
